# FrexTech AI Simulations API Documentation

## Overview

FrexTech AI provides a comprehensive API for generating, editing, and managing 3D worlds using state-of-the-art AI models. This API enables developers to create immersive virtual environments, edit existing scenes, and export to various formats.

**Base URL**: `https://api.frextech.ai/v1`

**API Version**: `v1`

**Content Type**: `application/json`

## Authentication

### API Keys

All API requests require authentication via API key. Include your API key in the request headers:

```http
Authorization: Bearer YOUR_API_KEY
